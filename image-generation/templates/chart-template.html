Chart.js Template

This template uses Chart.js for creating professional charts. Modify the data and options below.

Customization Guide

Chart Types

  • bar - Vertical or horizontal bars
  • line - Line graph for trends over time
  • pie - Circular chart for proportions
  • doughnut - Pie chart with center hole
  • radar - Multi-axis comparison
  • scatter - X-Y point plot

Color Palettes

// Professional Blues
backgroundColor: ['#2563EB', '#3B82F6', '#60A5FA', '#93C5FD']

// Warm Oranges
backgroundColor: ['#EA580C', '#F97316', '#FB923C', '#FDBA74']

// Multi-color Vibrant
backgroundColor: ['#EF4444', '#F59E0B', '#10B981', '#3B82F6', '#8B5CF6']

// Grayscale Professional
backgroundColor: ['#1F2937', '#4B5563', '#6B7280', '#9CA3AF']

Common Modifications

  • Change data.labels for X-axis categories
  • Change data.datasets[0].data for values
  • Change options.plugins.title.text for chart title
  • Change backgroundColor for colors
  • Adjust canvas width/height for different dimensions

Multiple Datasets

data: {
    labels: ['Q1', 'Q2', 'Q3', 'Q4'],
    datasets: [
        {
            label: 'Product A',
            data: [45, 62, 58, 71],
            backgroundColor: '#2563EB'
        },
        {
            label: 'Product B',
            data: [38, 54, 61, 66],
            backgroundColor: '#10B981'
        }
    ]
}
