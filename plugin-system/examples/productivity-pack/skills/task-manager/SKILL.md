# Task Manager Skill

This skill helps you manage tasks and to-do lists efficiently.

## Capabilities

- Create and manage tasks with priorities
- Set due dates and reminders
- Track task status (todo, in-progress, done)
- Organize tasks by project or category
- Generate task reports and summaries

## Usage

Ask Claude to:
- "Create a new task: implement user authentication"
- "List all high-priority tasks"
- "Mark task #5 as completed"
- "Show tasks due this week"

## Task Format

Tasks include:
- Title and description
- Priority (low, medium, high, critical)
- Status (todo, in-progress, done, blocked)
- Due date
- Tags and categories
- Dependencies

## Commands

- `create task` - Create a new task
- `list tasks` - Show all tasks
- `update task` - Modify an existing task
- `complete task` - Mark task as done
- `delete task` - Remove a task
