# Note Taker Skill

This skill helps you take and organize notes effectively.

## Capabilities

- Create structured notes in markdown
- Organize notes by topics and tags
- Search and filter notes
- Link notes together
- Export notes in various formats

## Usage

Ask Claude to:
- "Create a note about the meeting"
- "Find notes tagged with 'project-x'"
- "Link this note to the design document"
- "Export notes as PDF"

## Note Format

Notes support:
- Markdown formatting
- Code blocks
- Tables and lists
- Images and attachments
- Tags and categories
- Cross-references

## Organization

- Hierarchical folders
- Tag-based organization
- Full-text search
- Date-based filtering
