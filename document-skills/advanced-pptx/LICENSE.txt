pptx-v2 Enhanced PowerPoint Skill

This software provides enhanced PowerPoint (.pptx) generation and manipulation
capabilities with a modular architecture, improved performance, and full
backward compatibility with the original pptx skill.

Key Features:
- Modular, extensible architecture
- Enhanced inventory extraction and text replacement
- Improved layout engine with auto-fitting
- Extended media support (SVG, remote images)
- Comprehensive metadata and diagnostics
- Full backward compatibility with pptx-v1

This is an enhanced version of the original pptx skill, maintaining all
existing functionality while adding significant improvements and new
capabilities.
